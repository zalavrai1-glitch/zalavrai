📖 Guide de connexion étape par étape
1
Créer un compte Supabase gratuit
Rendez-vous sur supabase.com → cliquez Start your project → créez un compte avec Google ou email. C'est gratuit (plan Free suffisant pour ZALAVRAI).
2
Créer un nouveau projet
Dans le tableau de bord → New Project → choisissez le nom zalavrai → sélectionnez la région la plus proche (ex: Europe West) → définissez un mot de passe de base de données → cliquez Create new project. Attendez ~2 minutes.
3
Créer les tables — SQL Editor
Dans votre projet Supabase → menu gauche SQL Editor → New query → copiez-collez TOUT le contenu du bloc SQL ci-dessous (dans Réglages) → cliquez Run. Vous devriez voir Success.
✓ Ce SQL crée les 7 tables (utilisateurs, clients, ventes, paiements, stock, signalements, archives) + active RLS automatiquement
4
Récupérer l'URL et la clé
Dans votre projet → Project Settings (⚙️ en bas à gauche) → section API.

• Project URL : commence par https:// et finit par .supabase.co
• anon / public key : longue clé commençant par eyJ

⚠️ Utilisez bien la clé "anon" (publique), PAS la clé "service_role" (secrète)
5
Coller ici et connecter
Collez l'URL et la clé dans les champs en haut de cette page → cliquez Connecter & Synchroniser. Toutes les données locales seront envoyées vers Supabase et la synchronisation démarrera.
✓ Après connexion : tous les appareils connectés avec la même clé voient les mêmes données en temps réel

📚 GUIDE DE CONNEXION (4 ETAPES)
1
Creer le Google Sheets
▶ Aller sur sheets.google.com
▶ Cliquer + Nouveau pour creer un classeur vide
▶ Le nommer ZALAVRAI-DB
▶ Dans l'URL, copier l'ID (entre /d/ et /edit)
https://docs.google.com/spreadsheets/d/1ABC...XYZ/edit
2
Coller le Code.gs
▶ Dans le Sheets: menu Extensions → Apps Script
▶ Effacer le code existant
▶ Coller le contenu de Code.gs fourni
▶ Remplacer SHEET_ID par l'ID copie a l'etape 1
const SHEET_ID = '1ABC...XYZ';
▶ Sauvegarder avec Ctrl+S (ou ⌘S sur Mac)
3
Deployer comme Web App
▶ Bouton Deployer → Nouveau deploiement
▶ Cliquer ⚙️ → choisir Application Web
▶ Description: ZALAVRAI v1
▶ Executer en tant que: Moi
▶ Qui a acces: Tout le monde
▶ Cliquer Deployer → Autoriser l'acces
⚠ Choix Tout le monde requis pour que la PWA puisse se connecter
4
Coller l'URL et connecter
▶ Copier l'URL qui apparait apres le deploiement
https://script.google.com/macros/s/AKfy…/exec
▶ Coller l'URL dans le champ ci-dessus
▶ Cliquer Connecter & Synchroniser
▶ Puis cliquer Initialiser les feuilles Sheets
🔒 Securite: LockService anti-concurrence · Batch automatique · Outbox hors-ligne chiffree
🔄 Sync: Les donnees locales sont pushees vers Sheets a la connexion, puis tirees a chaque login
